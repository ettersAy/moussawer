# AI Incidents

This log captures bugfixes and project issues for AI/human sync.

## Format
- date
- mission-id
- issue
- resolution

### Example
- 2026-03-31 | MP-0001 | Missing `.ai` folder and docs | Added `.ai/context.md`, `.ai/constraints.md`, `.ai/conventions.md`, `.ai/incidents.md`, `.ai/mission-backend.md`, `.ai/mission-frontend.md`.
